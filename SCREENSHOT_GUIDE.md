# 📸 Screenshot Guide - Quick Reference

## Required Screenshots (6-8 images)

### 1. Main Interface (01-main-interface.png)
**What to capture:**
- Full browser window
- Demo mode warning visible (orange banner)
- Clean interface
- All 4 tabs visible

**How:**
- macOS: `Cmd + Shift + 4` → drag to select area
- Or: `Cmd + Shift + 3` for full screen

**Tips:**
- Close other tabs
- Make window full screen
- Clean, professional look

### 2. Research Query (02-research-query.png)

### 3. Sources Found (03-sources-found.png)

## Screenshot Checklist

Before taking screenshots:
- [ ] App is running (http://localhost:8501)
- [ ] Browser window is clean (close other tabs)
- [ ] Window is maximized or full screen
- [ ] Do 2-3 research queries to have data

While taking screenshots:
- [ ] High resolution (at least 1280x720)
- [ ] No personal information visible
- [ ] Clean, professional appearance
- [ ] All text is readable

After taking screenshots:
- [ ] Review each image
- [ ] Check for clarity
- [ ] Rename files properly (01-main-interface.png, etc.)
- [ ] Save in a "screenshots" folder

---

## macOS Screenshot Shortcuts

- `Cmd + Shift + 3` - Full screen
- `Cmd + Shift + 4` - Select area (recommended)
- `Cmd + Shift + 4` then `Space` - Capture window

Screenshots save to Desktop by default.

---

## Screenshot Sizes

**Devpost Requirements:**
- Format: PNG or JPG
- Max size: 5MB per image
- Min dimensions: 1280x720
- Recommended: 1920x1080

---

## Pro Tips

1. **Use consistent browser size** for all screenshots
2. **Hide bookmark bar** for cleaner look
3. **Use same query** across screenshots for consistency
4. **Take extras** - you can choose the best ones later
5. **Check lighting** - make sure screen is bright enough

---

## After Screenshots

1. Create a folder: `screenshots/`
2. Rename files properly
3. Review all images
4. Select best 6-8 for submission
5. Upload to Devpost
