## Description

### Inspiration
Research is overwhelming. Students, professionals, and academics spend countless hours sifting through articles, papers, and websites. We built Nova Research Agent to transform this tedious process into an intelligent, conversational experience powered by Amazon Nova AI.

### What it does
Nova Research Agent is an AI-powered research assistant that revolutionizes how people conduct research:

- **Intelligent Multi-Source Search**: Automatically gathers relevant information from multiple web sources
- **AI-Powered Analysis**: Uses Amazon Nova's advanced language models to analyze and understand content
- **Smart Synthesis**: Combines information across all sources into comprehensive, well-structured summaries with proper citations
- **Automated Insight Extraction**: Nova automatically identifies and highlights the most important findings
- **Visual Analytics Dashboard**: Track research patterns, topic trends, and activity over time with interactive charts
- **Interactive Q&A**: Answer follow-up questions with full conversation history
- **Professional Export**: Generate publication-ready reports in multiple formats (JSON, text, Markdown)
- **Source Transparency**: Every claim is backed by citations with relevance scoring

Instead of spending hours manually reading through dozens of articles, users simply ask a research question and receive a comprehensive, cited summary with visual insights in seconds.

### How we built it

**Technology Stack:**
- **Amazon Nova AI Models**: Core intelligence using Nova Pro via Amazon Bedrock
- **Python Backend**: Application logic with real algorithms for scoring and analysis
- **Streamlit**: Interactive web interface with 4 specialized tabs
- **Plotly**: Advanced data visualizations for analytics and insights
- **DuckDuckGo Search API**: Privacy-focused web search
- **BeautifulSoup4**: Intelligent content extraction

**Architecture:**
1. **Research Engine**: Searches web, fetches content, extracts main text
2. **Nova Client**: Interfaces with Amazon Bedrock for synthesis, insights, and Q&A
3. **Analytics Engine**: Processes research data with real algorithms
4. **Utility Functions**: Source relevance scoring, key term extraction, quality metrics

**Key Implementation:**
- Advanced prompt engineering for accurate Nova responses
- Real algorithms for source relevance scoring (not fake data)
- Robust error handling with helpful messages
- Demo mode for testing without AWS credentials
- Production-ready code with comprehensive documentation

### Challenges we ran into

1. **Source Relevance Scoring**: Developed real algorithms to calculate source quality based on query relevance, domain credibility, and content quality - no fake data!

2. **Content Extraction**: Web pages contain lots of noise. Implemented sophisticated extraction to get clean, relevant text while handling various site structures.

3. **Error Handling**: Created comprehensive error handling with helpful messages for AWS access issues, rate limiting, and network problems.

4. **Demo Mode**: Built a demo mode that works without AWS credentials, allowing testing of UI and algorithms while preparing for full Nova integration.

5. **Real-time UX**: Balanced thoroughness with speed using progress tracking and visual feedback.

- **Real Algorithms**: Implemented actual scoring algorithms for source relevance, key term extraction, and quality metrics
