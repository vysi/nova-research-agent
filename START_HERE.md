## 🚨 Common Issues & Solutions

### "AWS credentials not found"
**Solution**: Create .env file with your AWS credentials
```bash
cp .env.example .env
# Edit .env with your keys
```

### "No module named 'streamlit'"
**Solution**: Install dependencies
```bash
pip install -r requirements.txt
```

### "Access denied to Nova model"
**Solution**: Enable Nova in AWS Bedrock console
1. Go to AWS Bedrock
2. Click "Model access"
3. Request access to Nova Pro and Nova Lite
4. Wait for approval (usually instant)

### "No sources found"
**Solution**: Try a different query or check internet connection

**Read**: TROUBLESHOOTING.md for more solutions
