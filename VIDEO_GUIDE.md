# 🎬 Video Recording Guide

## Video Requirements

**Duration:** 2-3 minutes (max 4 minutes)
**Format:** MP4, MOV, or YouTube link
**Quality:** 1080p minimum
**Audio:** Clear narration

---

## Recording Setup (macOS)

### Option 1: QuickTime (Built-in, Recommended)
1. Open QuickTime Player
2. File → New Screen Recording
3. Click options:
   - Microphone: Built-in Microphone (or external)
   - Quality: Maximum
4. Click red record button
5. Select area or full screen
6. Click "Start Recording"

### Option 2: Cmd + Shift + 5 (macOS Mojave+)
1. Press `Cmd + Shift + 5`
2. Choose "Record Entire Screen" or "Record Selected Portion"
3. Click "Options":
   - Microphone: On
   - Show Mouse Clicks: On (recommended)
4. Click "Record"

## Recording Tips

### Before Recording:
- [ ] Close all other applications
- [ ] Close unnecessary browser tabs
- [ ] Turn off notifications (Do Not Disturb mode)
- [ ] Test microphone
- [ ] Practice the script 2-3 times
- [ ] Have app ready at http://localhost:8501

### During Recording:
- [ ] Speak slowly and clearly
- [ ] Pause between sections
- [ ] Show mouse movements clearly
- [ ] Don't rush
- [ ] If you make a mistake, pause and restart that section

### After Recording:
- [ ] Watch the full video
- [ ] Check audio quality
- [ ] Verify all features are shown
- [ ] Trim any dead space at start/end

## Video Editing (Optional)

### Simple Edits (macOS):
1. Open video in QuickTime
2. Edit → Trim
3. Drag yellow handles to trim
4. File → Export As → 1080p

### Advanced Edits:
- iMovie (free on Mac)
- DaVinci Resolve (free)
- Add title cards
- Add background music (optional, low volume)

## Upload to YouTube

### Steps:
1. Go to: https://youtube.com/upload
2. Click "Upload video"
3. Select your video file
4. Fill in details:

**Title:**
```
Nova Research Agent - Amazon Nova AI Hackathon 2026
```

**Description:**
```
Nova Research Agent - AI-powered research assistant using Amazon Nova

🔬 Features:
- Multi-source web search and synthesis
- AI-powered analysis with Amazon Nova
- Automated insight extraction
- Visual analytics dashboard
- Interactive Q&A
- Professional export options

🏆 Built for Amazon Nova AI Hackathon 2026

📦 GitHub: https://github.com/vysi/nova-research-agent
🌐 Devpost: [add link after submission]

#AmazonNova #AIHackathon #ResearchAssistant #AI #MachineLearning
```

**Visibility:**
- Choose "Unlisted" (can be viewed by anyone with link)
- Or "Public" (searchable on YouTube)

**Tags:**
```
Amazon Nova, AI Hackathon, Research Assistant, Artificial Intelligence, Machine Learning, AWS, Bedrock, Python, Streamlit
```

4. Click "Publish"
5. Copy the video URL

## Alternative: Loom

If you prefer easier recording:

1. Go to: https://www.loom.com
2. Sign up (free)
3. Install desktop app
4. Click "Start Recording"
5. Choose screen + camera (or just screen)
6. Record
7. Auto-uploads and gives you link

## Your Video URL

After uploading, your URL will be:
```

```

or

```

```

Use this URL for Devpost submission!
