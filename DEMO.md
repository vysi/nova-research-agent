### 3. Demo Flow

1. **Enter Query**: Paste one of the example queries
2. **Watch Progress**: Show the source gathering process
3. **Review Sources**: Expand the sources found section
4. **View Synthesis**: Navigate to Results tab to see AI-generated summary
5. **Ask Follow-up**: Go to Q&A tab and ask: "What are the main challenges?"
6. **Export**: Demonstrate export functionality

## Common Demo Questions

**Q: How is this different from ChatGPT?**
A: We specifically gather and cite multiple sources, then use Amazon Nova to synthesize them with proper attribution. It's research-focused, not just conversation.

**Q: What makes Amazon Nova special here?**
A: Nova's advanced language understanding allows for nuanced synthesis of complex information across multiple sources while maintaining accuracy and context.

**Q: Can it handle academic papers?**
A: Currently web sources, but PDF/academic paper support is on the roadmap!

**Q: How accurate are the results?**
A: We maintain source citations so users can verify information. Nova's synthesis is based on actual source content, not hallucination.
